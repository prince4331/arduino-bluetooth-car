# Bluetooth Car Controller

Serial Bluetooth interface that drives an L298N-based differential robot via simple commands.

## Original Folder
02_Robotics_and_Vehicles\Bluetooth_Car

## Hardware
Arduino, HC-05/HC-06 Bluetooth module, L298N bridge, dual DC motors, 12V battery pack.

## Notes
Tune PWM values inside Bluetooth_Car.ino to match the chassis torque requirements.
